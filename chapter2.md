# 对象 Objects
---

#### 对象字面量 Object Literals

包围在一对花括号中的零个或者多个“名/值”对，属性名不强制引号。
对象可以嵌套。

```
var empty_object = {};

var stooge = {
    'first_name': 'Jerry',
    'last_name': 'Huang'
}

var flight = {
    ariline: 'Oceanic',
    departure: {
        time: '2018-04-04',
        city: 'ShangHai'
    }
}
```
